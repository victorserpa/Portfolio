## 💻 Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ gh repo clone victorserpa/Portifolio
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm install

# Make sure the 'src/server/api.jsx' file has your API user

# Start the client
$ npm run dev
```
